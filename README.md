  1. 作业列表
     1. 添加作业

  1. 查看作业

        1. 纸质作业

     2. 班级作业
        1. 纸质作业【已交点击确认】
        2. 电子作业【上传文件】
  3. 作业提醒【短信、邮箱】
  4. 上传作业【上传1到多个文件】

     1. 显示提交与未提交的标识

  7. 修改作业
  8. 删除作业

* 班级管理

  1. 我的班级
  2. 修改班级

* 学生管理

  1. 学生列表
     1. 添加学生

  3. 删除管理
